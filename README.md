# PFO Frontend — Comparativa de Agentes IA · Coffee Shop
 
> Landing page generada por dos agentes de inteligencia artificial a partir de un mismo prompt, sin intervención humana posterior.

|---|---|

##  Prompt Utilizado
 
El siguiente prompt fue enviado de forma idéntica a ambos agentes:
 
```

"       Role:Act as a Senior Frontend Developer specialized in UX/UI, web accessibility, responsive design, SEO, and modern frontend development.

        Context:You must create a professional Landing Page for a fictional coffee shop called "Aurora Coffee House."
        The coffee shop offers specialty coffee, handcrafted pastries, and a warm, welcoming atmosphere where customers can relax, work, study, or enjoy time with friends.
        Before generating any code, carefully analyze all requirements and design the best possible solution by following frontend development, user experience, accessibility, performance, and visual design best        practices.

        Objective:Generate a modern, attractive, professional, and visually complete Landing Page that promotes Aurora Coffee House, conveying a warm, elegant, and inviting atmosphere that encourages visitors to stop by.

        Mandatory Functional Requirements:
        The Landing Page must include the following sections:

        1. Header
        Coffee shop name or logo.
        Navigation menu.
        Smooth navigation between sections.

        2. Hero Section
        Must include:

        An engaging headline.
        A short descriptive paragraph.
        A Call-To-Action (CTA) button such as "View Menu", "Reserve a Table", or "Visit Us".
        A visually appealing coffee-related image or illustration.

        3. About Us
        Include:
        Coffee shop story.
        Philosophy.
        Commitment to quality coffee and handcrafted products.

        4. Coffee Shop Specialties
        Include at least three featured categories or products.

        Examples:
        Specialty Coffee.
        Handcrafted Pastries.
        Vegan Options.
        Breakfast.
        Brunch.

        Each item must contain:

        Title.
        Description.
        Icon or representative visual element.

        5. Testimonials
        Include at least three fictional customer testimonials.
        Each testimonial must include:
        Customer name.
        City.
        Review.

        6. Contact Form
        Must include:
        Name.
        Email.
        Message.
        Submit button.

        Backend functionality is not required.

        7. Footer
        Must include:
        Copyright.
        Social media links.
        Business hours.
        Address.
        Additional navigation links.
        Design Requirements
        Modern, elegant, and professional design.
        Clean and minimalist appearance.
        Use light, warm, and visually pleasant colors.
        The visual style should be inspired by a cozy contemporary coffee shop.
        Excellent readability.
        Modern and professional typography.
        Effective use of whitespace.
        Consistent visual language across the entire page.
        Intuitive and user-friendly interface.
        Clear visual hierarchy.
        Incorporate images or illustrations related to coffee, beverages, and pastries to enhance the visual experience.
        Accessibility and Usability

        The interface must follow UX and accessibility best practices:

        WCAG 2.1 AA compliance.
        Proper text-to-background contrast.
        Keyboard navigation support.
        Semantic HTML structure.
        Accessible forms.
        Descriptive labels.
        Visible hover and focus states.
        Appropriate font sizes.
        Intuitive experience for all users.
        Responsive Design

        The page must adapt correctly to:
        Mobile devices.
        Tablets.
        Desktop screens.

        Components should automatically reorganize according to screen size.
        Basic SEO

        Implement:
        Meta title.
        Meta description.
        Semantic HTML structure.
        Proper heading hierarchy.
        Basic search engine optimization best practices.
        Performance
        Optimized code.
        Fast loading times.
        Avoid unnecessary elements.
        Maintain an efficient and maintainable structure.
        Technical Requirements
        Use only HTML5, CSS3, and Vanilla JavaScript.
        Do not use React, Angular, Vue, Next.js, or any frontend framework.
        Do not implement backend functionality.
        Do not use databases.
        Do not use external dependencies requiring installation.
        Use Flexbox and/or CSS Grid.
        Maintain a clear separation between structure (HTML), styling (CSS), and behavior (JavaScript).
        Use semantic HTML elements.
        Include brief comments in the main sections of the code.
        Restrictions
        Generate the entire project autonomously.
        Do not request additional information.
        Do not leave any section incomplete.
        Do not use placeholder text such as "Lorem Ipsum."
        Generate realistic content consistent with the coffee shop theme.
        The final result must be ready to run without manual modifications.
        Deliverables

        Generate all files required to run the project:
        index.html
        style.css
        script.js

        Ensure all files are complete, properly linked, and immediately runnable."
```
